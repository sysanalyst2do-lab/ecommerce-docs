# Схемы базы данных

Этот раздел содержит схемы и модели данных для e-commerce системы.

## Структура

- ER-диаграммы
- SQL схемы (DDL)
- Миграции базы данных
- Описание моделей данных

## Статус

Раздел в разработке. Схемы БД будут добавлены по мере проектирования.

## Планируемые схемы

- PostgreSQL схемы для основных доменов
- Схемы для поискового индекса (Elasticsearch)
- Схемы событий (Kafka message schemas)
