# API спецификации

Этот раздел содержит спецификации API для e-commerce системы.

## Структура

- REST API спецификации (OpenAPI/Swagger)
- GraphQL схемы (если применимо)
- Событийные API (Kafka topics и схемы сообщений)
- Интеграционные API (внешние системы)
