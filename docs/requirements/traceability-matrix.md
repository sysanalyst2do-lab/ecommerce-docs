# Traceability Matrix

## 1. Overview

Данный документ фиксирует трассировку между бизнес‑требованиями (BR) и системными требованиями (SR) для e‑commerce системы. [web:6][web:10]

## 4. Notes and Usage

- Для тест‑кейсов можно завести отдельную таблицу `SR ID → Test Case IDs`, дополнительно к этим матрицам. [web:7][web:13]
- Рекомендуется поддерживать эту матрицу в актуальном состоянии через pull request‑ы при добавлении/изменении BR и SR. [web:10][web:16]
