# Руководство по развёртыванию

## Окружения

### Development
- Локальная разработка с Docker Compose
- Автоматическая перезагрузка при изменениях

### Staging
- Тестовое окружение для QA
- Максимально приближено к Production

### Production
- Продакшн окружение
- Высокая доступность и мониторинг

## Развёртывание

### Backend

```bash
# Сборка Docker образа
docker build -t ecommerce-backend:latest ./backend

# Развёртывание
docker-compose up -d backend
```

### Frontend

```bash
# Сборка production версии
npm run build

# Развёртывание статических файлов
# (nginx, CDN, или другой статический хостинг)
```

## Миграции БД

```bash
# Применение миграций
alembic upgrade head

# Откат миграций (при необходимости)
alembic downgrade -1
```

## Конфигурация

- Использовать переменные окружения для конфигурации
- Secrets хранить в secure vault (например, HashiCorp Vault)
- Не коммитить секреты в репозиторий

## Мониторинг

- Health checks для всех сервисов
- Логирование в централизованную систему
- Метрики производительности

## Откат (Rollback)

- Поддержка быстрого отката к предыдущей версии
- Версионирование миграций БД
- Blue-green или canary deployments

## Статус

Раздел в разработке. Детальные инструкции будут добавлены по мере настройки инфраструктуры.
